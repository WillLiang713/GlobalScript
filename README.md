# GlobalScript

这是一个集合了 Clash 规则配置、自动化脚本及相关文档的项目。旨在通过集成化的脚本和精细化的规则管理，提升代理网络的使用体验。

## 项目结构

*   `GlobalScript.js`: **核心脚本**。用于动态生成 Clash 配置，包含 DNS 设置、分流规则、自动筛选节点及地区分组等功能。配合 Clash Verge (Rev) 等客户端使用效果更佳。
*   `Clash_Full.yaml` / `Clash_Full.ini`: 预设的完整 Clash 配置文件/初始化文件。
*   `docs/`: 包含 ACL4SSR 规则参考和 Subconverter 使用帮助。
*   `demo/`: 存放示例配置，方便快速上手。

## 主要功能

1.  **动态分组**: 脚本会自动根据节点名称识别地区（如香港、美国、日本等）并创建对应的延迟测试分组。
2.  **精细分流**: 内置了 AI、开发工具、社交媒体、游戏平台等多维度的分流策略。
3.  **高性能规则集**: 采用 `.mrs` 规则集格式，显著降低客户端加载负担。
4.  **智能 DNS**: 配置了国内外分流导向的 DNS 解析，减少 DNS 污染并加速访问。

## 快速使用

1.  如果你使用支持脚本处理的 Clash 客户端（如 Clash Verge Rev），可以将 `GlobalScript.js` 的内容引入作为预处理脚本。
2.  根据需要修改 `GlobalScript.js` 顶部的 `customRules` 来添加你自己的个性化规则。
3.  确保你有可用的代理节点列表（Proxy Provider 或 Proxies）。
