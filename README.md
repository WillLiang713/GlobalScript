# GlobalScript

精简强大的 Clash 代理配置方案，一份配置搞定所有分流需求。

## 核心文件

- **Clash_Full.yaml** - 完整的 Clash 配置文件，开箱即用

## 功能特性

- ✅ **智能分流** - 涵盖 OpenAI、Gemini、GitHub、Netflix 等主流服务
- ✅ **地区分组** - 自动按香港、美国、日本、新加坡、台湾、韩国等地区分类节点
- ✅ **自动测速** - 各地区节点自动进行延迟测试并选择最优线路
- ✅ **ACL4SSR 规则** - 采用社区广受好评的 ACL4SSR 规则集

## 快速开始

1. 下载 `Clash_Full.yaml` 到你的 Clash 客户端配置目录
2. 导入你的订阅节点或手动添加节点
3. 启动 Clash 即可使用

## 支持的服务

**AI 平台**: OpenAI、Gemini  
**开发工具**: GitHub、Microsoft  
**流媒体**: Netflix、YouTube、国外媒体  
**社交通讯**: Telegram、Twitter  
**游戏平台**: Steam、Epic、Origin、Sony、Nintendo
